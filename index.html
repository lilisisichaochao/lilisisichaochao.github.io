神经网络
神经网络由输入层、输出层、中间层（隐藏层）构成。一般网络命名是以权重所在层数或者共有几层进行命名。
激活函数。H(x)会把输入的信号总和转换为输出信号，这种函数一般称为激活函数，激活函数的作用在于决定如
何来激活输入信号的总和。例如a=b+w1x1+w2x2,y=h(a),a为计算加权输入信号和偏置的总和，然后用h()激活
函数将a转化为输出y.激活函数是连接感知机和神经网络的桥梁。一般的“朴素感知机”是指单层网络，指的是激活
函数使用了阶跃函数的模型，“多层感知机”是指神经网络，即使用sigmoid函数等平滑的激活函数的多层网络。
实际上，进入神经网络的大门，关键在于将阶跃函数换为其他函数。

sigmoid函数

                                                1
   sigmoid函数：h(x)=______________________
                                         1+exp(-x)

exp(-x)表示e的-x次幂，e是纳皮尔常数2.7182.

通过感知机和神经网络函数的比较，可以得知，感知机神经元流动的是0或1的二元信号，而神经
网络中流动的是连续的实数值信号。阶跃函数和sigmoid函数在平滑性上有一定的差距，但是可以
发现他们具有相似的形状。实际上，两者结构均是“输入小时，输出接近0（为0），随着输入增大，输出接近1（为1）”
也就是说，但输入信号为重要信息时，两个函数都会输出较大的值，当输入信号为不重要信息时，两
者都会输出叫小的值，还有一个共同点，不管输入信号有多笑，或者有多大，输出的值都在0到1之间。

非线性函数
神经网络的激活函数必须使用非线性函数，因为使用线性函数，神经网络的层次就没有意义了。
线性函数的问题在于，不管如何加深层数，总是存在与之等效的“无隐藏层的神经网络”。例如
线性函数h(x)=cx作为激活函数，把y=h(h(h(x)))的运算作为三层神经网络，这个运算会进行y=ccc*x
的运算，同样也可以由y(x)=c的3次幂来表示，无法发挥层次优势。

ReLU函数 |x (x>0)
ReLU函数的输入大于0时，直接输出该值，在输入小于等于0时，输出0。h(x)= |0 (x<=0)

感知机
感知机由美国学者Frank在1957年提出。感知机是学习通向神经网络深度学习的重要思想。
1、感知机是什么
感知机接收多个输入信号，输出一个信号。感知机的信号会形成流向前输出信号，感知机的信
号有“流/不流”（1/0）两种取值。每个输入信号送往神经元（节点）的过程中会乘以对应的
w1,w2。。。权重。当神经元计算传输过来的信号总和，当超过某个界限，就会输出1，称为
“神经元被激活”，这个界限称为阈值，用特定符号表示。感知机每个输入信号都有各自固有
的权重，这个权重都发挥着控制各个信号的作用。也就是说，权重越大，对应权重的信号的重
要性也就越高。
2、简单逻辑电路
与门。与门是两个输入一个输出的门电路，与门仅在两个输入都是1时输出1，其余都为零。参数
选择方法（0.5，05，0.7）且有无数种组合。
与非门。与非门和与门是两个完全相反的门电路，与非门当两个输入都是1时为零，其余
都为1.参数组合（-0.5，-0.5，-0.7）即和与门取相反数。
或门。或门只要一个输入为1，输出就是1。
3、感知机的实现
现在，利用python来实现刚才的逻辑电路，先定义一个接收参数x1,x2的AND函数
def AND(x1,x2):
w1,w2,theta=0.5,0.5,0.7
tmp=x1w1+x2w2
if tmp<=theta:
return 0
elif tmp>theta:
return 1
然后初始化函数就可以得到想要的效果
将theta换为-b得到b+w1x1+w2x2<=0,其中b为偏置，w1,w2为权重
实现权重和偏置：def AND(x1,x2):
x=np.array([x1,x2])
w=np.arry([0.5,0.5])
b=-0.7
tmp=np.(w*x)+b
if tmp<=0:
return 0
else:
return 1
4、感知机的局限性
异或门。异或门也被称为逻辑异或电路，仅当输入的x1,x2的一方为1时输出才会是1。
由前面可知异或门是不能被实现的。根据权重和阈值组成的方程画图可以得到，一条
一条直线无法把输出1和0的区间正确的分开，所以不能实现。
线性和非线性。当权重图中输入不同结果的区间不能被正确的分开，此时可以把“直线
”这个限制条件去掉。因此可知感知机的局限性就在于它只能表示由一条直线分割的空间
，而用曲线分开的不能用感知机正确表示。曲线分割而成的空间称为非线性空间，由直线
分割而成的空间称为线性空间。
多层感知机。感知机的绝妙之处就在于它可以叠加层。叠加了多层的感知机也称为多层
感知机。

numpy的梯度函数
np.gradient(a) ： 计算数组a中元素的梯度，f为多维时，返回每个维度的梯度
离散梯度： xy坐标轴连续三个x轴坐标对应的y轴值：a, b, c 其中b的梯度是（c-a）/2
而c的梯度是： (c-b)/1

当为二维数组时，np.gradient(a) 得出两个数组，第一个数组对应最外层维度的梯度，第二个数组对应第二层维度的梯度。
